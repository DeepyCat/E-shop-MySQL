Toto je nejlepší E-Shop kdo kdy viděl.
Tady je jak by se měl udělat aby to viděli úplně všichni, i vy.

  1. v phpMyAdmin vytvořte databází s názvem e_shop

  2. Import e_shop.sql do databáze e_shop

  3. Do db.php to upravte aby to fungovalo pro vás

  4. Toto potom otevřít http://localhost/index.php
